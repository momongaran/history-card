# 知的図鑑UI 微調整 最終仕様（vFinal）

## ■ 基本方針

* 装飾を削る
* 情報階層を視覚強度で表す
* 線で区切らない
* 色は識別補助にとどめる
* レーン構造は「静かな整列」

---

# ① 因果要素表示方式

## ● バッジは廃止する

従来の：

* 背景色
* 枠線
* 角丸
* 点線
* 強い色面

はすべて削除。

---

## ● 表示はテキスト主体（span）に統一

```
B01   B02   F03   R01   UNC00
```

のような横並びテキスト表示。

---

# ② 配置ルール

### ● 垂直位置

* イベント行の中央に揃える
* baseline揃えは禁止
* line-heightで中央制御

```
display: flex;
align-items: center;
```

---

### ● バッジ間隔

* 全要素の gap は **8px固定**
* 行ごとの詰まり禁止
* 可変余白は使わない

---

### ● B / F / R の境界処理

* 線は使わない
* 区切り線は禁止
* 境界のみ margin-left を追加

例：

* Fブロックの前に +12px
* Rブロックの前に +12px

これにより「空気の区切り」をつくる。

---

# ③ 色と視覚階層

## ● 背景色は完全廃止

色面による塊感は出さない。

---

## ● 文字色で階層を作る

| 要素    | 表示強度 |
| ----- | ---- |
| F     | やや濃い |
| B     | 標準   |
| R     | 標準   |
| UNC00 | 薄灰   |

---

### 推奨例

```
B / R  → #444
F      → #222
UNC00  → #bbb
```

※彩度は使わない
※識別目的の最小限の差のみ

---

# ④ UNC00 の扱い

* 表示は維持
* サイズは他と同一
* 点線は使わない
* 背景なし
* 薄灰色テキストのみ

視覚階層を一段下げるだけ。

---

# ⑤ 左右カラム整理

### ● 境界処理

* 強い線は禁止
* 1pxの薄線 or 24px以上の余白

推奨：

```
border-right: 1px solid #eee;
```

または

```
margin-right: 32px;
```

---

# ⑥ 全体トーン

* 白背景固定
* ダークモード禁止
* グラデーション禁止
* 影（box-shadow）禁止
* 立体感禁止

「紙面」のような静けさを優先。

---

# ⑦ まとめ（最終決定事項）

✔ 背景色廃止
✔ 枠線廃止
✔ 角丸廃止
✔ 点線廃止
✔ spanテキスト表示
✔ gap 8px固定
✔ B/F/R境界は余白のみ
✔ Fだけやや濃い文字色
✔ UNC00は薄灰文字
✔ 線は極薄のみ可

1. Freeze方針

  • UI機能追加なし（静的描画維持）
  • レイヤ: C / F / P / R 維持
  • F（揺らぎ）は必須1

2. v2.2追加事項

  • frameworkView.meta.backgroundInheritance を許容
  • frameworkView.meta.backgroundDerivation を許容
  • 背景辞書は作らない（核と具体を分離）

3. 継承リンク形式

{
  "backgroundInheritance": [
    {
      "fromEventId": "EV_prev",
      "backgroundRefs": ["EL_prev_C_01"],

    }

}

4. 原則

  • 事後評価をカテゴリ基準に使わない
  • 背景同一性は操作的に柔軟
  • UI互換を壊さない

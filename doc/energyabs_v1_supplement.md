# EnergyAbs v1 補足仕様書

> 更新日: 2026-03-20
> 対象バージョン: EnergyAbs v1（energyabs_v1.json + slot_taxonomy.json）
> 本文書は EnergyAbs モデルのUI実装に伴い導入された分類体系と画面仕様を記述する。

## 1. fwType（意味的FW型）

ノードの因果パターンを8種に分類する。各型にアイコン（Unicode記号）・色を割り当て、UI上で一貫して使用する。

| fwType | shape | color | label | 意味 |
|---|---|---|---|---|
| battle | ⚔ | #c45a3c | 武力衝突 | 武力による決着 |
| conflict | ▲ | #c8960a | 対立蓄積 | 緊張・対立の蓄積 |
| collapse | ⊘ | #8a3a3a | 崩壊 | 体制・秩序の崩壊 |
| institution | ■ | #3a6a9a | 制度化 | 制度・ルールの確立 |
| power | ◆ | #6a3a8a | 権力確立 | 権力・体制の形成 |
| shock | ⚡ | #c47a3a | 外生衝撃 | 外部からの衝撃・導入 |
| drift | 〜 | #5a8a5a | 漸進変化 | 緩やかな構造変化 |
| reform | ✦ | #3a7a8a | 改革 | 意図的な変革 |

### 1.1 意味領域（Semantic Domains）

fwTypeを4つの意味領域にグルーピングする。

| 領域 | 含まれるfwType | 色 | 意味 |
|---|---|---|---|
| 地面 | drift, institution, collapse | rgba(139,119,90,.25) | 構造的な地盤 |
| 力学 | conflict, battle, power | rgba(196,90,60,.25) | アクター間の力の作用 |
| 意図 | reform | rgba(58,122,138,.25) | 意図的な介入 |
| 外部 | shock | rgba(196,122,58,.25) | 外部からの作用 |

### 1.2 fwType遷移

エッジは fwType(source) → fwType(target) の遷移を表す。93エッジから45種の遷移型が観測される。頻度上位: drift→drift(7), drift→conflict(5), conflict→battle(5), institution→institution(4)。

## 3. UI画面構成

### 3.1 因果フレーム図鑑（demo_catalog.html）

インスタンスビュー。EnergyAbsグラフの全ノードをクラスタ→深度順にブロック表示。

- **ブロック**: 枠色=fwType色、内部にref-chip（入力元）・ラベル・概要・抽象パターン・アクター・出力先
- **SVG線**: ref-chip⇔ソースブロック間の同一性リンク（点線）
- **コントロール行1**: roleフィルタ（amplifies/triggers/transforms/sustains）、凡例トグル
- **コントロール行2**: 意味領域フィルタ、fwTypeフィルタ、逆引き検索
- **凡例**: フローティング、デフォルト非表示

### 3.2 抽象パターン図鑑（demo_pattern_catalog.html）

パターンビュー。エッジをfwType遷移型でグルーピングし、抽象パターンチェーンとして表示。

- **セクションヘッダ**: fwType遷移型（例: ⚡→◆）、パターン数・件数
- **パターンカード**: fw-icon（26px角、色付き、ホバーでラベル）→ パターン文（{スロット}にslot-icon付き）
- **インスタンス折りたたみ**: 各パターンの具体的ノードペア一覧
- **フィルタ**: fwType別絞り込み

### 3.3 スロット分類一覧（demo_slot_taxonomy.html）

分類ビュー。128語のスロット語彙を7大分類・27小分類で一覧表示。

- **サマリテーブル**: 大分類×小分類の語数・使用回数
- **カテゴリセクション**: SVGアイコン付きヘッダ
- **用語カード**: クリックで使用箇所展開（該当スロットをハイライト）

### 3.4 画面間リンク（未実装）

3画面間の相互遷移は今後の課題。

## 5. 型レベル再集約の分析結果（v1時点の予備調査）

### 5.1 スロット型シグネチャによる集約

70個のユニークパターンをスロットの大分類型の並びだけで集約すると **36シグネチャ** になる。上位6シグネチャで全体の49%（34/70件）をカバー。

| シグネチャ | 件数 | 例 |
|---|---|---|
| ●主体→□構造 | 11 | {勝者}が{主導体制}を確立する |
| ●主体→●主体 | 10 | {新秩序派}が{旧秩序派}を武力排除する |
| ●主体→◇力学 | 4 | {指導者}の死が{後継争い}の空白を生む |
| ●主体（単独） | 3 | {地方}が叛乱を起こす |
| ●主体→≈変動 | 3 | {王権}が{対外関与}の伝統を持続する |
| ▬資源→▬資源 | 3 | {人口}が{制度の容量}を超過する |

### 5.2 作用型（動詞の方向性）による二次集約

スロット型シグネチャが同じでも動詞の方向性が異なるケースがある（例: 「確立する」vs「打倒する」）。動詞を作用型に正規化することで、実質 **20〜25個の集約パターン** に収まる見込み。

作用型の候補（6〜8種）:

| 作用型 | 含まれる動詞 | 例 |
|---|---|---|
| 獲得 | 確立/掌握/樹立/完成/独占/建設 | ●が□を獲得する |
| 破壊 | 打倒/解体/崩壊/排除/滅ぼす | ●が□を破壊する |
| 蓄積 | 高まる/激化/拡大/超過 | ◇が蓄積する |
| 維持 | 持続/継承/定着 | ●が≈を維持する |
| 衝撃 | 揺さぶる/襲う/仕掛ける | ●が□に衝撃を与える |
| 逆転 | 自立化/凌駕/迂回 | ●が●を逆転する |
| 動員 | 動員/利用/呼応 | ●が●を動員する |
| 変容 | 転換/侵食/形骸化/乖離 | ▬が▬を変容させる |

### 5.3 次バージョンの実装方針

1. 作用型を確定し `slot_taxonomy.json` に追加
2. 各パターンに `typeSignature`（スロット型並び）と `actionType`（作用型）を付与
3. パターン図鑑を集約パターンで再グルーピング
4. 世界史展開時に再分類を見直す前提で設計

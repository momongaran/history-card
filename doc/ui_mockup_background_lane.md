# 背景レーン UIモックアップ（v3.9検討用）

> 題材: 律令的土地制度の変質過程（701〜1069）

## レイアウト構成

```
[イベントレーン]  ← タイムライン上のイベント
[因果レーン]      ← E → F → B（イベント側からB側へ）
[背景レーン]      ← プロセスのライフサイクル
```

## 背景レーンの読み方

- 横線（━━━）がプロセスのライフサイクル
- ●が生成点（701: 大宝律令でプロセス駆動開始）
- ▶が現在も変質中（または別プロセスに置換されるまで）
- 各イベントのB要素（因果レーン最下段）が、背景レーンのプロセスとの接続点
- プロセス上の状態ラベル（安定運用→変質開始→変質加速→形骸化）がライフサイクルの段階

## ポイント

1. **律令的土地制度の変質**と**荘園制の拡大**が逆相関で動いている
2. 一方のプロセスの変質が他方のプロセスを駆動する関係が視覚的に見える
3. 各イベントのB要素が「どのプロセスのどの段階の断面か」が縦の接続線でわかる
4. 1069年の延久荘園整理令は、律令側からの「立て直し」とも、荘園側への「抑制」とも読める
